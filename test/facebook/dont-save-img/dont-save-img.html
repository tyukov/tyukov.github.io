Don't save image tag — простой но крайне понятное решение от Facebook. Аналогичное решение видел на Apple Music, единственной дифференциацией было width: 100%; height: 100%; вместо top: 0; bottom: 0; left: 0; right: 0.
Стандартным способом (через правую кнопку мыши) изображение скачать не получится.
